# ⚡ NEXUS AI — Live Intelligence Hub

An AI-powered live assistant with real-time web search,
4 AI models and 5 personas. Powered by Groq API.

## 🌟 Live Demo
👉 [Try it here!](https://huggingface.co/spaces/BUDDDY2894830/nexus-ai-assistant)

## ⚡ Features
- 🌐 Real-time web search
- 🤖 4 AI Models — LLaMA 3.3 70B, LLaMA 3.1 8B, Mixtral, Gemma
- 🎭 5 Personas — Assistant, Researcher, Coder, Writer, Coach
- 🧠 Conversation memory
- ⚡ Ultra fast — Groq powered

## 🛠️ Tech Stack
- Python
- Groq API
- Compound Beta — built-in web search
- Gradio — Web Interface
- Hugging Face Spaces — Deployment

## 🚀 Run Locally
pip install groq gradio
python app.py

## ⚠️ Setup
Add your Groq API key in app.py or set
GROQ_API_KEY as environment variable
